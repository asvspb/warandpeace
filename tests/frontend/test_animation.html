Тест анимации фона бейджа

Этот тест показывает, как работает анимация изменения фона при клике:

Описание функции:

  • При клике на бейдж "Нет" добавляется CSS-класс "muted"
  • Класс "muted" активирует анимацию "pulse"
  • Анимация плавно меняет фон от синего (#007AFF) к прозрачному
  • Цвет текста также изменяется для лучшей видимости

Текущий CSS-код анимации:

.badge.muted {
    animation: pulse 1s infinite alternate;
}

@keyframes pulse {
    from { background-color: #007AFF; color: white; }
    to { background-color: transparent; }
}
